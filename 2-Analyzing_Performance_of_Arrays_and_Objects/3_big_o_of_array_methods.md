# Big O of Array Built-in Methods

Please keep in mind that we don't need to memorize this information, it's just a good reference:

-  push - O(1)
-  pop - O(1)
-  shift - O(n)
-  unshift - O(n)
-  concat - O(n)
-  slice - O(n)
-  splice - O(n)
-  sort - O(n * log n)
-  forEach/map/filter/reduce/etc - O(n)
